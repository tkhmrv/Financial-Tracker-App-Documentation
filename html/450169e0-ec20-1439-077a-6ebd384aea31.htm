      ExpensesForm - методы

      Методы

      ClearFields Очищает все поля ввода.
      DataGridViewExpenses_CellClick Обработчик события для выбора строки в таблице расходов.
      DisplayExpensesCategories Отображает категории расходов.
      DisplayExpensesData Отображает данные о расходах.
      Dispose Clean up any resources being used.
      (Переопределяет ContainerControlDispose(Boolean))
      Expenses_buttonAdd_Click Обработчик события для добавления нового расхода.
      Expenses_buttonClear_Click Обработчик события для очистки полей ввода.
      Expenses_buttonDelete_Click Обработчик события для удаления существующего расхода.
      Expenses_buttonUpdate_Click Обработчик события для обновления существующего расхода.
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      RefreshData Обновляет данные на форме расходов.
      ValidateInput Проверяет корректность введенных данных.
