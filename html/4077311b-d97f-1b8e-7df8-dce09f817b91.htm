      ButtonAddCategory_Click Обработчик события клика на кнопку добавления категории.
      ButtonDashboard_Click Обработчик события клика на кнопку панели приборов (dashboard).
      ButtonExpenses_Click Обработчик события клика на кнопку расходов.
      ButtonIncome_Click Обработчик события клика на кнопку доходов.
      ButtonLogout_Click Обработчик события клика на кнопку выхода из системы.
      DisplayUsername Отображает имя пользователя, вошедшего в систему.

      (Переопределяет FormDispose(Boolean))
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      InitializeDash Инициализирует панель приборов (dashboard).
      LabelCloseApp_Click Обработчик события клика на метку закрытия приложения.
