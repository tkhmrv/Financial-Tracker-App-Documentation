      IncomeForm - методы

      Методы

      ClearFields Очищает все поля ввода.
      DataGridViewIncome_CellClick Обработчик события для выбора строки в таблице доходов.
      DisplayIncomeCategories Отображает данные о доходах.
      DisplayIncomeData Отображает категории доходов.
      Dispose Clean up any resources being used.
      (Переопределяет ContainerControlDispose(Boolean))
      Income_buttonAdd_Click Обработчик события для добавления нового дохода.
      Income_buttonClear_Click Обработчик события для очистки полей ввода.
      Income_buttonDelete_Click Обработчик события для удаления существующего дохода.
      Income_buttonUpdate_Click Обработчик события для обновления существующего дохода.
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      RefreshData Обновляет данные на форме доходов.
      ValidateInput Проверяет корректность введенных данных.
