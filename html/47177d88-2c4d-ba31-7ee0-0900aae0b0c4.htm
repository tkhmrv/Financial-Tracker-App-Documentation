      CategoryUserControl - методы

      Методы

      Category_buttonAdd_Click Обработчик события клика по кнопке добавления категории.
      Category_buttonClear_Click Обработчик события клика по кнопке очистки полей.
      Category_buttonDelete_Click Обработчик события клика по кнопке удаления категории.
      Category_buttonUpdate_Click Обработчик события клика по кнопке обновления категории.
      ClearFields Очищает поля ввода.
      DataGridViewCategory_CellClick Обработчик события клика по ячейке в DataGridView.
      dataGridViewCategory_CellFormatting Красит строки в разные цвета, в зависимости от типа категории.
      DisplayCategoryList Отображает данные категорий.
      DisplayCategoryListStatic Отображает данные категорий, используя статический экземпляр пользовательского интерфейса.
      Dispose Clean up any resources being used.
      (Переопределяет ContainerControlDispose(Boolean))
      InitializeComponent Required method for Designer support - do not modify the contents of this method with the code editor.
      RefreshData Обновляет данные на форме.
      ValidateCategoryInputs Проверяет корректность введенных данных для категории.
